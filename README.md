# Equipe 6 | Logística & Cadeia de Suprimentos

Apresentação interativa em React sobre capacidade de resposta, modelos Push/Pull,
ponto de desacoplamento e estratégias híbridas na gestão logística da cadeia de
suprimentos.

## Como rodar

Instale as dependências:

```bash
npm install
```

Abra em modo desenvolvimento:

```bash
npm run dev
```

Gere a versão final:

```bash
npm run build
```

Pré-visualize o build:

```bash
npm run preview
```

## Navegação

- Setas direita/esquerda, espaço, PageUp/PageDown, Home e End.
- Swipe horizontal em telas touch.
- Menu responsivo no mobile.

## Checks de entrega

```bash
npm run lint
npm run build
```
